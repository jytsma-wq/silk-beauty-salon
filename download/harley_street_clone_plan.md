## 🎨 Design System

### Color Palette

```
Primary Colors:
- Deep Black/Navy: #1a1a2e or similar dark tone
- White: #ffffff
- Gold/Accent: #c9a962 or similar elegant gold

Secondary Colors:
- Light Gray: #f5f5f5
- Medium Gray: #e0e0e0
- Dark Gray: #333333

Text Colors:
- Primary Text: #1a1a2e
- Secondary Text: #666666
- Light Text: #999999

Button Colors:
- Primary CTA: Gold/Warm tone
- Secondary: Dark outline
```

### Typography

```
Headings: Playfair Display (serif, elegant)
- H1: 48-64px, elegant serif
- H2: 32-40px
- H3: 24-28px

Body Text: Poppins (sans-serif)
- Regular: 16-18px
- Small: 14px

Navigation: Poppins Medium
- 14-16px
```

### Spacing & Layout

```
Container Width: 1200px (max)
Page Padding: 4vw
Header Padding: 1.3vw vertical
Section Spacing: 60-100px between sections
```

### Buttons

```
Primary Button:
- Background: Gold/warm accent
- Text: White or dark
- Border Radius: 0 (sharp corners) or minimal
- Padding: 15px 30px
- Font: Uppercase, letter-spacing

Secondary Button:
- Background: Transparent
- Border: 1-2px solid dark
- Text: Dark
```

### Animations

```
- Smooth scroll behavior
- Fade-in animations on scroll
- Hover effects on cards and buttons
- Slide-up content reveals
- Animation duration: 0.6s
- Animation style: ease
```

## 📱 Responsive Breakpoints

```
Desktop: > 1200px
Tablet: 768px - 1200px
Mobile: < 768px

Mobile Adjustments:
- Hamburger menu
- Stacked cards (1-2 columns)
- Smaller typography
- Reduced padding
```
